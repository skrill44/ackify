# Privacy Policy

Ackify respects user privacy and collects only the data required for its core functionality.

---

## What data is stored

Ackify stores:

* Server IDs
* Channel IDs
* Message IDs related to announcements
* User IDs only for tracking acknowledgment and checklist completion

Ackify does **not** store message content beyond what is needed to display or reference announcements and checklist items.

---

## What data is not collected

Ackify does not collect:

* Email addresses
* IP addresses
* Personal profile information
* Direct messages
* Message history outside of Ackify features

---

## How data is used

Data is used only to:

* Track acknowledgments
* Track checklist items and completion
* Provide command functionality

Data is never sold, shared, or used for advertising.

---

## Data retention

Data remains stored only as long as it is needed for the feature it supports.
Server owners can request deletion of stored data at any time.

---

## Contact

If you have questions or concerns about privacy, contact the Ackify team through the official support channel.
